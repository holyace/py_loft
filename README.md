# py_loft

>基于python的抓包demo，模仿LOFTER的App，发起api请求，抓取用户喜欢的图片列表，然后下载图片。原版App有些图片无法下载，就算能下载，也会在图片上添加水印。
通过本demo下载的图片没有水印(前提是非转载的图片，转载的图片会被加上水印)。
